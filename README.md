# AGP Закупки Бот

Telegram бот для Меруерт — управление закупками и оплатами через Google Sheets.

## Что умеет

- 📎 Принимает Excel выгрузку с оплатами → обновляет Мастер-таблицу
- 📷 Распознаёт фото платёжных поручений через Claude OCR
- 📊 /report — сводка по всем договорам
- 🔍 /find [номер] — найти конкретный договор
- ⚠️ /expiring — договора истекающие через 30 дней
- ✅ Подтверждение перед каждой записью (защита от ошибок)
- 📝 Лог всех платежей в отдельном листе

## Установка

### 1. Клонируй репозиторий на GitHub
Залей всю папку agp_bot на GitHub (приватный репозиторий).

### 2. Заполни .env
```
TELEGRAM_TOKEN=токен от BotFather
ALLOWED_CHAT_ID=chat_id Меруерт (узнать через @userinfobot)
SPREADSHEET_ID=ID Google Sheets таблицы
ANTHROPIC_API_KEY=твой ключ Anthropic
GOOGLE_CREDS_JSON=credentials.json
```

### 3. Добавь credentials.json
Положи JSON файл сервисного аккаунта Google в папку рядом с bot.py.
Назови его credentials.json.

### 4. Поделись таблицей
В Google Sheets → Поделиться → добавь email сервисного аккаунта
(sheets-bot@agp-zakupki-bot.iam.gserviceaccount.com) с правами редактора.

### 5. Деплой на Render
1. render.com → New → Blueprint → подключи GitHub репо
2. Заполни переменные окружения в дашборде
3. В Secret Files добавь credentials.json
4. Deploy!

## Структура Google Sheets

Мастер-таблица должна иметь листы:
- "Мастер" — основная таблица договоров
- "Лог платежей" — история всех платежей от бота

## Команды бота

| Команда | Действие |
|---------|----------|
| /start | Приветствие и список команд |
| /report | Сводка: оплачено/частично/ожидается |
| /expiring | Договора истекающие через 30 дней |
| /find AGP/GEN/TS-01/2025 | Найти конкретный договор |
| Отправить .xlsx файл | Массовое обновление оплат |
| Отправить фото | OCR распознавание платёжки |
